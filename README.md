## Online Pet Store System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/4eceeb6cfa0645d292b36db79fbe1306/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Functional Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Pet Boarding
5. Products
6. Forum
7. Announcement Information
8. Personal Center
9. Delivery Address
10. Pet Boarding Orders
11. Boarding Logs
12. Product Favorites
13. Product Reviews
14. Product Orders
15. Shopping Cart
16. Online Customer Service


**Administrator Functions:**
1. Login
2. Personal Center (change password, personal information)
3. Administrator Management
4. User Management
5. Product Management
6. Pet Boarding Management
7. Pet Boarding Order Management
8. Boarding Log Management
9. Customer Service Chat Management
10. Basic Data Management (pet type management, boarding service type management, product type management, announcement type management)
11. Forum Management
12. Announcement Information Management
13. Carousel Management
